# Requirement

* Python 3.8.5
* Flask==1.1.2
* gunicorn==20.0.4
* line-bot-sdk==1.16.0
* matplotlib==3.3.2
* numpy==1.18.5
* opencv-python-headless==4.2.0.32
* Pillow==7.2.0


# Installation

```bash
pip install -U Flask
pip install gunicorn
pip install line-bot-sdk
pip install matplotlib
pip install numpy
pip install opencv-python-headless
pip install Pillow
```
